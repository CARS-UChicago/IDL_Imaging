GSECARS Imaging Software

This document describes the GSECARS-developed IDL software used for 2-D and 3-D image display. All of these IDL routines are available in a tar file.

  • image_display.pro is a general purpose 2-D image display tool. It can be used to display slices in through 3-D data sets.
  • volume_render.pro is a GUI-based 3-D volume rendering tool. It permits setting cutting planes, modifying color and opacity tables, etc.
  • make_movie.pro is a procedure for displaying 3-D data as a movie on the screen or to write it to a disk file in MPEG format.
